RTPGUI

A configurable RTP (Random Teleport) plugin designed for Minecraft practice servers.

✨ Features

* 🎮 Simple and intuitive RTP GUI
* ⚙️ Fully configurable
* 🌍 Support for teleporting in specific worlds
* 🚀 Lightweight and optimized for practice servers

📋 Commands

Command	Description
/rtp	Opens the RTP GUI
/rtp <world>	Randomly teleports you in the specified world

⚙️ Configuration

RTPGUI is fully configurable, allowing you to customize its behavior to match your server’s needs.
