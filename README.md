## Features
- Configurable RTP system
- Simple GUI interface
- Support for world-specific RTP
- Lightweight and easy to configure
- Built for practice servers
---
## Commands
| Command | Description |
|---------|-------------|
| `/rtp` | Opens the RTP GUI |
| `/rtp <world>` | Randomly teleports the player in the specified world |
---
## Configuration
RTPGUI is designed to be fully configurable, allowing you to customize worlds, messages, GUI settings, and teleport behavior to fit your server.
